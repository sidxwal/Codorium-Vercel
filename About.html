About the Website

Codorium

Hello Engineers, Welcome to Codorium, this website is created by me, Siddhant Khandelwal as my Frontend Project in Web Development. This website is very useful for fresher engineering students who want to get placed in IT Sector but they don't have proper direction to follow and end up wasting their valuable time searching for everything.

This website contains many resources for students to help them in their 1st or 2nd year when they are not aware about skilling up and also dont have any guidance to follow. You will get :

  • Free Resources to learn any programming language or any technology.
  • Well researched step by step roadmap for any Job Profile apart from this you will not need to search anywhere.
  • Bunch of Project Ideas to learn any technology deeply because project based learning is very important.
  • Detailed information and references for any Job Profile to know about them.
  • Various chat groups to discuss and network with your fellows which are Learning Pods.
  • Finally an Extras page which contains all the remaining helpful resources that can help you.

If these are not enough which I know are not then you can tell me anything you want this website should contain in the Suggestions Box in Homepage of Website. Thank You for reading till here ❤
