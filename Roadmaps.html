ROADMAPS

On this page you'll get proper step by step roadmap for learning various languages and tools for different job profiles. You can trust the Roadmaps given in this page 100% because they are very well researched from various sources. The resources to learn all these technologies are provided on another page.

Full Stack Web Development

➼ Start Frontend
Learn HTML, CSS, Javascript


This is the most essential part which will make your foundation strong in Web Development Field.

➼ Make Projects and Learn Frameworks
Make Projects like Clones of Popular Websites or Your Self Portfolio
Learn CSS Frameworks like Bootstrap or Tailwind
Also Javascript Framework of React.JS


Making Projects will make you learn more effectively than just Watching Tutorials and and We should learn different Frameworks to make our work more productive and Easier.

➼ Start Backend
Learn Node.JS, Express.JS and MongoDB


Node.JS is primarily used in Backend Development and allows you to run Javascript on server side, outside of web browser while Express.JS is a framework of Node.JS and it helps Developers build Web Applications more easily. MongoDB is basically a type of Database that is designed to store and manage data. You can know more about these Technologies on Google.

➼ Essentials
Now as your MERN Stack (MongoDB, Express.JS, React.JS, Node.JS) will be completed which is Full Stack now its time to learn some extra essentials :
Learn Basic Terminal Commands and understand Git & Github

Git is tool which will help you keep track of changes in your code and it helps when you are working on any project with your friend. Github is a website you can store your code, it's like a cloud storage service for your code projects, it makes easier to share your code with anyone or collaborate on Projects.

➼ Finals
Now as you have learned almost everything that is necessary in Full Stack Web Development then there are 2 things left :

Make 2-3 End to End Full Stack Projects which you can add in your Portfolio and Resume
You also need to learn Deployment of your Website which you'll make out of your learnings.

Progress: 0%

Android Development

➼ Master a Programming Language
Learn Java or Kotlin


You should choose Kotlin here because it has more concise syntax than Java. You have to spend your time Understanding Kotlin and you have to focus more on Logic Building.

➼ Understand Version Controlling (Git)
Learn and Understand Git & Github


When you are building an App and doing serious development then it is necessary to learn Git you can learn more about Git and Github on Google.

➼ Using Different Tools
Learn to use Android Studio and Android Virtual Studio


Android Studio is a tool that gives complete environment for Android Development like code editor, testing features, debugging tools which makes easier for developers to design, build and improve their apps. AVS is a tool to mainly test apps on computer without needing a physical android device and you can also test your app in different phone models and android versions.

➼ Make an App
Start by making a Basic app like any Music App or Tic Tac Toe App and then make a Complex app like News Reading App or E-Commerce App. This will help you test your knowledge practically and build a confidence.

This much is sufficient for getting a Beginner Level Job in Android Development and here are some things you must not avoid during this journey :
➼ Focus on Basics and get Expertise.
➼ Don't just watch Tutorials but also make some Apps at your own for Practical Knowledge.
➼ Not gaining experience by taking any beginner level job and just learning everything.

There are many more things in Android Development which you will learn while job or making projects.

Progress: 0%

Software Engineer

➼ Learn a Programming Language
Learn Python or Java or C++


You should choose between Java and C++ because in Indian Placements, Python isn't considered that much as compared to C++ and Java. You would have to start learning basics and you have to not jump to another language without getting well in first.

➼ Explore Data Structures and Algorithms
Excel DSA in C++ or Java


DSA tells us how to approach a problem step by step and how to organize the data in an efficient data structure. In most Job's Technical Interviews there are DSA Rounds where candidates are asked to resolve any problem.

➼ Practice Problem Solving
Master in Problem Solving


Problem Solving is a skill which you can acquire by practising problems and understand different patterns associated with them. You can practice Problem Solving from Leetcode, Codechef and Hackerrank.

➼ Build a Project
So now to apply the skills you learned and getting hands on experience, you will have to make a Project.


To start you can make simple - ToDo List, TicTacToe or a Library Management System and after that you can start building more complex projects.

➼ Start looking for Internships and Job Opportunities

After learning all the necessary skills, now its time to look for Internships (2nd, 3rd Year) and Job Oppurtunities (4th Year) to get Hnads On Industry Experience.

Progress: 0%
