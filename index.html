Some Useful Websites

These websites will help you a lot during your Coding Journey. Every coder must have an account on these sites. These websites are totally free to use.

GitHub

It is like Facebook of Coders and here we can create, store, merge and collaborate on files or our Projects. The main use of this platform is to allow developers to host thier files in a 'Git Repository' so that other people can collaborate on Projects with them.

Stack Overflow

It is a Question and Answer website for Computer Programmers, here you can post Questions regarding any error in your code and people here will Answer your Question and anyone can Upvote or Downvote the Answer hence it is very useful for specially Beginners.

GeeksforGeeks

This website offers tutorials, articles on various CS Topics such as DSA, DBMS, OS. Here you will also get a wide range of Interview Preparation materials including Coding Questions, Mock Interviews and more and you can also get vast collection of coding challenges by difficulty level.

W3Schools

It is a place where you can learn various anything about various web technologies such as HTML, CSS, Javascript, PHP, Python, SQL, Java, C++, C and many more and you can use it as a quick reference guide for developers.
